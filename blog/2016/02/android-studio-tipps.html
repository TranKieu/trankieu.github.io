Android Studio Tipps

Die Tipps für Android Studio, um effizienter zu arbeiten.

Hotkey

Android Logcat Color

Die Farbe für Logcat wie folgt ändern:

File -> Settings -> Editor -> Colors & fonts -> Android Logcat

Deaktivieren: Use inherited attributes und background

  • Warning: #EC971F
  • Verbose: #E6E6E6
  • Info: #31B0D5
  • Error: #C9302C
  • Debug: #449D44
  • Assets: #3071A9

Split Screen

Right-clicking on Tab -> Split Vertically
Add Hokey for Split Vertically 
File -> Settings -> Editor -> Keymap
  • Search for Split Vertically.
  • Double Click and select Add Keyboard Shortcut.
  • Add Ctrl + Shift + Y .
